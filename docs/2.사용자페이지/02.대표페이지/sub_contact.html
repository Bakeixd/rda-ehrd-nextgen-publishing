찾아오시는길

농촌인적자원개발센터

(54874) 전주특별자치도 전주시 덕진구 농생명로 420 농촌인적자원개발센터

승용차
  • 경부고속도로 대전방면
  • 한남IC (경부고속도로 대전방면) → 서울TG → 신갈 → 기흥 → 동탄 → 안성 → 망향 → 천안JC (천안논산고속도로 세종방면) → 풍세 → 정안 → 북공주JC (광주방면) → 공주1JC (남공주방면) →탄천 → 남논산TG → 논산JC (익산방면) → 여산 → 익산JC (삼례방면) → 서전주IC (동김제,이서방면) →서전주TG (순창,전주방면) → 농촌인적자원개발센터
  • 호남고속도로, 익산장수고속도로 : 서전주IC(전주방면) 출차일 경우
  • 서전주IC → 콩쥐팥쥐로 → 안전로 → 농생명로 → 정여립로 → 정문
  • 서전주IC → 콩쥐팥쥐로 → 정여립로 → 정문
  • 순천완주고속도로 : 상관IC(전주방면) 출차일 경우
  • 상관IC → 21번 국도(서전주IC 방면) → 콩쥐팥쥐로 → 안전로 → 농생명로 → 정여립로 → 정문
  • 상관IC → 21번 국도(서전주IC 방면) → 콩쥐팥쥐로 → 정여립로 → 정문

시외버스
  • 전주시 시외버스터미널 출발
  • 택시전주시외버스터미널 → 혁신도시 입구(우회전) → 정여립로 약 1km 전방 U턴 → 정문(약200m)
  • 버스전주시외버스터미널 → 446m이동(금암광장 청담한방병원) 165번 승차→ 두현마을 하차(45분 소요) → 뒤로 100m이동 → 횡단보도 건넌 후 정문

기차
  • 전주역 출발
  • 버스동부대로 전주역(버스정류장) 210번 승차 → 두현마을 하차(1시간 소요) → 횡단보도 건넌 후 정문
